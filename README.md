# Customer Demographics Study

This project analyzes customer demographic data and visualizes the distribution of customers across different age groups using Python.

## Tools Used

- Python
- Pandas
- Matplotlib

## Features

- Load customer demographic data from a CSV file
- Analyze customer age groups
- Visualize demographic distribution using a bar chart

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the script:

python customer_demographics.py

## Output

The project generates a chart named `customer_demographics.png` showing customer distribution by age group.
